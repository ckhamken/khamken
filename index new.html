Things to consider

  • We utilize a Formlabs Form 2 stereolithography printer which has a maximum build volume of 145 x 145 x 175mm. If your object has features that are larger than these dimensions, we ask that you split it into multiple segments that can be later assembled together. If needed, we can assist in performing this at an additional charge.
  • Currently, we offer four(4) standard colors (clear, white, grey, black) and two(2) functional materials (flexible, tough). By default we use a 50 microns print resolution which offers a balance of superb quality and fast print times.
